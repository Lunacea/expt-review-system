## 3. 実験方法
被験者に対し、Webベースの実験システムを用いて以下の実験を行う。なお、実験1Aおよび1Bは、被験者ごとにランダムな順序で実施されるタスクを含む（カウンターバランス）。

### 実験1A：数値的評価との比較
Amazonの実データを用いた5件のレビューに対し、以下の2つの手法で評価を行う。
- **タスクA**: 数値的評価（5段階★評価）
- **タスクB**: 選択式アノテーション（提案手法）
- **事後アンケート**: 両手法の比較（表現力、容易性など）

### 実験1B：テキスト評価との比較
同様に5件のレビューに対し、以下の2つの手法で評価を行う。
- **タスクA**: テキスト評価（自由記述）
- **タスクB**: 選択式アノテーション（提案手法）
- **事後アンケート**: 両手法の比較（入力負荷、表現力など）

### 実験2：数値的評価の質的確認
数値的評価の限界を確認するためのアンケート調査。

## 4. データ収集・分析計画
- **ログデータ**: 評価所要時間、アノテーション数、修正回数など。
- **アンケート**: リッカート尺度による主観評価。
- **比較検証**:
    - 数値的評価 vs 提案手法：具体性の向上度合い。
    - テキスト評価 vs 提案手法：入力時間の短縮効果、心理的負荷の低減。

## 5. 使用データ
- **レビューデータ**: Amazonレビューデータセット（`amazon_reviews_test.jsonl`）より抽出した、多様な評価・長さを持つ5件のレビューを使用する。
