# 実験システム 利用マニュアル (Ver 2.0)

## システム概要
本システムは、オンラインカスタマーレビューに対する新しい評価手法（選択式アノテーション）の有効性を検証するための実験プラットフォームです。実際のAmazonレビューデータを用いて評価を行っていただきます。

## アクセス方法
1. 指定されたURLにアクセスします。
2. 管理者から配布された「被験者ID」を入力してログインしてください。

## 実験の流れ

### 1. 同意書の確認
- ログイン後、実験同意画面が表示されます。PDFの説明書を確認し、署名を行ってください。

### 2. 実験の実施（ダッシュボード）
- 実験一覧画面には、あなたに割り当てられた実験が表示されます。
- **表示されている順番（上から順）** にタスクを進めてください。
  - ※ 比較実験の順序（数値的評価が先か、選択式評価が先かなど）は被験者ごとにランダムに設定されています。

### 3. タスクの操作
各タスクでは、**5件のレビュー** に対して連続して評価を行います。

#### A. 数値的評価タスク
1. レビュー文を読みます。
2. 「役に立った」かどうかを5段階（★）で評価します。
3. 「次のレビューへ」ボタンを押し、5件すべて評価したら完了です。

#### B. テキスト評価タスク
1. レビュー文を読みます。
2. そのレビューに対する評価・感想をテキストボックスに入力します。
3. 5件すべて入力したら完了です。

#### C. 選択式アノテーションタスク（提案手法）
1. レビュー文が表示されます。
2. **評価したい文（センテンス）をクリック** します。
3. 表示されるメニューから、当てはまるリアクション（「わかりやすい」「共感できる」など）を選択します。
   - **複数選択が可能** です。
   - 一度選択したリアクションをもう一度押すと解除できます。
4. 必要な評価をすべて行ったら、「次のレビューへ」を押して進みます。

#### D. アンケート
- 各実験の区切りで、比較評価に関するアンケートが表示されます。率直なご意見をお聞かせください。

### 4. 実験の終了
- 全てのタスクが完了すると、ダッシュボードに「全ての実験を終了する」ボタンが表示されます。

## 注意事項
- ブラウザの「戻る」ボタンは使用せず、画面内のリンクを使用してください。
- タスク完了後、次のタスクに進むボタンが有効にならない場合は、一度画面を更新（リロード）してみてください（不具合修正済みですが念のため）。
